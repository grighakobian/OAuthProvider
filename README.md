## Installation

### CocoaPods

OAuthProvider is available through [CocoaPods](https://cocoapods.org). To install
it, use the following entry in your Podfile:

```ruby
pod 'OAuthProvider'

# or 

pod 'OAuthProvider/RxSwift'

# or

pod 'OAuthProvider/ReactiveSwift'

```

Then run `pod install`.
